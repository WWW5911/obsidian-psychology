#### 健康生活包含五大層面：

- ================
- | 情緒 <-> 想法 // / / / |
  | ↑↓ ↗↙ ↑↓ 環境 /|
  | 生理 <-> 行為 / / / / /|
 ===============
- 這五個環節互相都有關聯，如：
	- 行為改變會影響思考方式與感受 (包含生理與情緒)
	- 行為也會改變環境與生活事件
	- 想法改變也會影響行為、情緒、生理反應，可能會造成環境變化

> 人會變老，我所有朋友很快會一個個死去 (想法)
> 所以不再與朋友敘舊 (行為)
> 與朋友疏離後，自己感到孤獨與悲傷 (情緒)
> 導致失眠與疲勞問題 (生理)
> 既然不與別人聯絡，朋友也不再打電話給自己 (環境)
